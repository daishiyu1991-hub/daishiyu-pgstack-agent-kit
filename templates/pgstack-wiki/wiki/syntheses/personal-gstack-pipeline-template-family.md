# Personal GStack Pipeline Template Family

## Shared Lifecycle

```text
collect or discover
-> confirm or filter
-> triage
-> synthesize
-> deliver
-> persist
-> operate
-> retro
```

## Starter Templates

- Daily brief
- Radar pipeline
- Research sidecar

## Reference Chain

```text
Radar Hub -> AI Daily Brief -> NotebookLM Research Sidecar -> PGBrain
```

## Runtime Split

- Hermes runs recurring bounded steps.
- Codex designs, tests, and evolves the rules.
- PGBrain stores durable knowledge.
