# Overview

This starter wiki is a local `PGStack Node`.

It gives a teammate:

- a workflow layer for durable tasks
- a brain layer for durable knowledge
- a deterministic local engine
- Codex and Hermes integration points
- optional connection to a MemTensor team hub

The first reference chain is:

```text
Radar Hub -> AI Daily Brief -> NotebookLM Research Sidecar -> PGBrain
```
