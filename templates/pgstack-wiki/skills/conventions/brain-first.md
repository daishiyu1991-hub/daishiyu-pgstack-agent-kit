# PGStack Brain-First Convention

Before external research or new rule creation, check what PGStack already knows.

## Protocol

1. Read the relevant repo-level skill.
2. Query `PGBrain Engine` for matching objects.
3. Read the most relevant wiki or brain pages.
4. Check local/team MemTensor only as governed supporting memory.
5. Use external sources only for a real gap or freshness requirement.

## Rule

External research fills gaps. It does not replace the local operating brain.
