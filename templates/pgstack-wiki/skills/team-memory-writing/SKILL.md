# Team Memory Writing

Use this skill only after a memory passes the team gate.

## Contract

Write team memory as compact reusable operating knowledge, not as transcript
storage or personal configuration.

## Allowed Curated Team Trunks

- `business`
- `ai_knowledge`
- `skills`

## Rules

- Write compiled reusable knowledge, not chat logs.
- Include scope, source of truth, review status, and why it matters.
- Prefer pointers to wiki/brain pages over long standalone prose.
- Keep personal configuration, host-specific residue, and colloquial process
  chatter out of `team_shared`.
