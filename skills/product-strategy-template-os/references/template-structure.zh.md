# 固定模板结构

本文件是默认模板的章节边界。每章必须先生成执行计划，再采集证据、写分析、渲染完整报告，并停在人类决策。

## 1. 品类本质小结

目标：定义品类满足什么需求，以及价值金字塔。

模块：

- 产品人群：基础用户画像信息。
- 产品场景：产品主次使用场景。
- 使用用途：主要用来满足什么需求。
- 需求矩阵：职能价值、情感价值、改变生活、社会影响。

输出：

- 品类到底在解决什么问题。
- 用户为什么会搜索或购买。
- 进入第 2 章需要补什么市场证据。

## 2. 市场竞争分析

目标：判断市场容量、趋势、生命周期、Amazon 渠道和其他渠道机会。

模块：

- 2.1 市场大盘：全球主要市场大盘、市场发展趋势、市场生命周期。
- 2.2 亚马逊渠道分析：关键词搜索趋势、市场容量&趋势、品牌集中度、市场新品趋势。
- 2.3 其他渠道市场机会：Walmart、Target、HomeDepot、Lowe's 等渠道。

输出：

- 品类是否值得继续。
- 行业阶段。
- 渠道优先级。
- 进入第 3 章要拆哪些品牌和竞品。

## 3. 头部品牌竞争&竞品分析

目标：判断头部品牌基因、能力模型、产品矩阵和扩维打法。

模块：

- 3.1 头部品牌矩阵：品牌定位、能力、资源范围、战略建议。
- 3.1.x 品牌拆解：公司简介、用户定位、技术能力、渠道、销售、社媒、供应链公开线索。
- 3.2 产品竞争矩阵：头部产品、产品矩阵、核心参数对比。

输出：

- 短线主要竞争对象。
- 长线学习对象。
- 产品差异化入口。

## 4. 用户场景&需求分析

目标：把用户是谁、在什么场景使用、购买决策和真实需求拆清楚。

模块：

- 4.1 用户画像：地域、个人信息、兴趣、经济水平、用户特性总结。
- 4.2 场景矩阵：使用场景、场景关联分析。
- 4.3 用户旅程：用户旅程图、购买决策链。
- 4.4 需求分析：需求挖掘、listing 评论分析、高价值评论集、产品需求集。
- 4.5 产品需求&技术创新：创意挖掘、众筹平台线索。

输出：

- 核心用户场景。
- 需求优先级。
- 第 5 章营销表达和第 7 章产品规划的输入。

## 5. 营销分析&社媒传播

目标：判断对手运营水平、站外承接、社媒突破点和内容打法。

模块：

- 5.1 亚马逊平台运营竞争分析：品牌主页、listing 设计水平、SKU 产品布局。
- 5.2 其他平台运营竞争分析：Walmart 等渠道运营。
- 5.3 独立站：品牌直营渠道和站外承接能力。
- 5.4 社媒传播：官方组织、展会、论坛、媒体、话题集、网红投放。

输出：

- 亚马逊运营差距。
- 其他平台运营机会。
- 独立站能力差距。
- 社媒传播机会。
- 第 7 章上市策略输入包。

## 6. 供应链管理

目标：判断成本结构、供应链可行性、工艺优势和供应路径。

模块：

- 6.1 上游供应链分析：核心组成部件、头部供应商特征、主要客户、优劣势。
- 6.2 供应链竞对分析：竞品成本对标、关键部件拆解、工艺差异、模具/工装、组装成本。

输出：

- 核心成本控制点。
- 供应链可行性。
- 竞品成本差异。
- 我们可能的成本/工艺优势。
- 第 7 章产品规划输入包。

## 7. 产品规划

目标：把前面章节的用户、竞争、供应链和营销判断投射到产品定义、listing 策略和销售计划。

模块：

- 7.1 产品定义全流程：需求收敛、功能/参数、取舍清单。
- 7.2 listing 策略+设计策划：购买决策树、图片主题排序、A+/视频/五点描述主题。
- 7.3 销售计划：营销规划、强攻主市场或细分人群突入、KOL 投放。

输出：

- 产品定义。
- listing 策略。
- 设计策划。
- 销售计划。
- 上市验证计划。
